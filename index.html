HTML & CSS

HTML (HyperText Markup Language) :

  • Créer une structure de base pour une page web.
  • Utiliser des balises sémantiques pour structurer le contenu.
  • Insérer des liens hypertexte et des images.
  • Utiliser les balises de liste pour créer des listes ordonnées et non ordonnées.

CSS (Cascading Style Sheets) :

  • Appliquer des styles visuels à une page HTML en utilisant des règles CSS.
  • Utiliser des sélecteurs pour cibler des éléments spécifiques et leur appliquer des styles.
  • Contrôler la mise en page en utilisant des propriétés CSS telles que la largeur, la hauteur, le positionnement, les marges et les rembourrages.
  • Appliquer des styles de texte, tels que la couleur, la taille de police, la police de caractères, l'espacement des lettres et des mots.

Java

  • Écrire des programmes Java en utilisant les concepts de base tels que les variables, les types de données, les opérateurs et les structures de contrôle.
  • Manipuler des tableaux et des collections de données.
  • Comprendre et utiliser les principes de l'orienté objet, tels que les classes, les objets, l'héritage et le polymorphisme.
  • Implémenter des interfaces et utiliser des classes abstraites.
  • Gérer les exceptions et les erreurs dans les programmes Java.
  • Utiliser des bibliothèques standard de Java pour des tâches courantes, telles que l'entrée/sortie, la manipulation de fichiers et les opérations sur les chaînes de caractères.
  • Créer des interfaces graphiques simples en utilisant des bibliothèques telles que JavaFX ou Swing.
  • Comprendre les concepts de base de la programmation réseau en Java, tels que les sockets et les protocoles de communication.
  • Utiliser des outils de développement Java tels que l'environnement de développement intégré (IDE) Eclipse ou IntelliJ.
